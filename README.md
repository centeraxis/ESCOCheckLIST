# 催事運営チェックリスト

催事運営の手順を管理するためのWebアプリケーションです。

## 機能

- 日常業務フローのチェックリスト
- チケット配り・配置の管理
- 混雑・イレギュラー対応のガイド
- 進捗状況のトラッキング
- 報告書の作成とエクスポート（コピー、LINE、メール、印刷）

## 技術スタック

- **フレームワーク**: React 18
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS
- **アイコン**: Lucide React
- **デプロイ**: Vercel

## セットアップ

### 必要な環境

- Node.js 18以降
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build

# ビルドのプレビュー
npm run preview
```

## Vercelへのデプロイ

1. GitHubリポジトリにプッシュ
2. Vercelダッシュボードで「New Project」を選択
3. GitHubリポジトリを接続
4. 自動でビルド設定が検出されます
5. 「Deploy」をクリック
